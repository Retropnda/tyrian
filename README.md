# tyrian
juego de naves de los 90s

Tyrian es un matamarcianos de desplazamiento vertical para MS-DOS desarrollado por World Tree Games Productions y publicado en 1995 por Epic MegaGames. El juego fue oficialmente lanzado como freeware en 2004, y los gráficos estuvieron hechos disponibles bajo una licencia abierta en abril de 2007.
